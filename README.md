# ethanr.co.uk

This is the source code for my [personal website](https://ethanr.co.uk).

Hosted on AWS S3, served with AWS Cloudfront.

Bootstrapped with [Create React App](https://github.com/facebook/create-react-app)

### Develop Locally
```npm run start```

### Build
```npm run build```

### Deploy to S3
```npm run deploy```
